# Creazione modelli

# Namirial

Entrare sulla piattaforma Namirial e andare su:

  • Nuovo Documento
  • Carica
  • Inserire il destinatario (e eventuali altri destinatari in copia conoscenza o in richiesta di presa visione)
  • Se si vuole inserire l'autenticazione tramite messaggio cliccare sul lucchetto e abilitare "codice SMS"
  • Impostare l'oggetto del messaggio (se diverso da quello già impostato) e il corpo del messaggio (se diverso da quello già impostato)
  • prosegui
  • impostare i campi sul modulo

Impostare campi per landing page

Per la compilazione tramite landing page inserisci nel campo “id” un nome adatto al campo, tutto in minuscolo senza spazi e caratteri speciali. Usare _ per sostiture gli spazi

  • deselezionare "Permetti la delega al destinatario"
  • click su DEV
  • click su JSON V4

File Json

Il file Json scaricato ti servirà per caricarlo su SmartSender

# SmartSender

  • Fai Click sull'avatar con le iniziali del tuo nome e cognome in alto a destra
  • Modelli Namirial
  • Crea Modello
  • Selezione l'account a cui vuoi associare il modello
  • Inserisci il nome del modello
  • Carica il file .json scaricato in precedenza
  • Carica il documento pdf (non obbligatorio)

WARNING

Il documento è obbligatorio quando c'è un invio automatico (Es: Invio tramite landingpage), se l'operatore deve inviare e ricaricare il documento in fase di invio allora non è necessario caricarlo

  • associa il modello a un'organizzazione
  • carica
